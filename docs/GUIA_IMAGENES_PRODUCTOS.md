## Especificaciones Técnicas

| Característica | Requisito |
|----------------|-----------|
| **Proporción** | **1:1 (cuadrado)** |
| **Tamaño mínimo** | 1200 x 1200 píxeles |
| **Tamaño ideal** | 2000 x 2000 píxeles |
| **Formato** | JPG, PNG o WEBP |
| **Peso máximo** | 1MB por imagen |
| **Fondo** | Blanco puro (#FFFFFF) |

## ¿Por qué cuadrado (1:1)?

```
✅ CORRECTO (1:1)              ❌ INCORRECTO (16:9)
┌────────────────┐             ┌────────────────────────┐
│                │             │                        │
│   ┌────────┐   │             │  ┌────────┐            │
│   │        │   │             │  │  PROD  │            │
│   │ PRODUCT│   │             │  │  ucto  │ espacios   │
│   │        │   │             │  └────────┘ vacíos     │
│   └────────┘   │             │                        │
│                │             └────────────────────────┘
└────────────────┘             La imagen no aprovecha el espacio
Todo el espacio usado
```

## Reglas del Producto en la Imagen

### ✅ El producto debe verse COMPLETO

```
┌────────────────────────┐
│       margen           │
│    ┌──────────────┐    │
│    │              │    │
│ m  │   PRODUCTO   │ m  │  ← Producto centrado
│ a  │   COMPLETO   │ a  │  ← Con espacio "respirando"
│ r  │              │ r  │
│    └──────────────┘    │
│       margen           │
└────────────────────────┘

El producto debe ocupar ~70-80% del espacio
Dejar ~10-15% de margen en todos los lados
```

### ❌ NUNCA cortar el producto

```
INCORRECTO:
┌────────────────────────┐
│────────────────────────│ ← Producto cortado arriba
│         │              │
│ PRODUCTO│              │
│ CORTADO │              │
│────────────────────────│ ← Producto cortado abajo
└────────────────────────┘
```

## Fotos Sugeridas para la Galería Principal

| # | Tipo de Foto | Descripción | Ejemplo Mental |
|---|--------------|-------------|----------------|
| 1 | **Principal/Frontal** | Vista frontal completa | La "cara" del producto |
| 2 | **Lateral** | Vista de perfil | Muestra profundidad |
| 3 | **Perspectiva 3/4** | Ángulo diagonal | Vista más dinámica |
| 4 | **Trasera** (si aplica) | Parte de atrás | Conexiones, acabados |
| 5 | **Detalle 1** | Acercamiento a textura | Material, acabado |
| 6 | **Detalle 2** | Acercamiento a mecanismo | Bisagras, uniones |
| 7 | **Escala** (opcional) | Con objeto de referencia | Entender tamaño |
| 8 | **Video** (opcional) | Producto en movimiento | Funcionalidad |

## Especificaciones Técnicas

| Característica | Requisito |
|----------------|-----------|
| **Formatos aceptados** | PDF, XLSX, XLS, DOCX, DOC, ZIP |
| **Peso máximo** | 10MB por archivo |
| **Nombre de archivo** | Descriptivo, sin espacios |

## Archivos Recomendados por Producto

| Prioridad | Archivo | Contenido Sugerido |
|-----------|---------|-------------------|
| ⭐⭐⭐ | **Ficha técnica** | Medidas, materiales, peso, acabados |
| ⭐⭐⭐ | **Manual de montaje** | Instrucciones paso a paso |
| ⭐⭐ | **Certificados** | PEFC, calidad, garantía |
| ⭐⭐ | **Guía de cuidados** | Mantenimiento y limpieza |
| ⭐ | **Planos/CAD** | Para arquitectos/diseñadores |

## Nomenclatura de Archivos

### ✅ Correcto
```
ficha-tecnica-mesa-comedor-roble.pdf
manual-montaje-estanteria-modular.pdf
certificado-pefc-2024.pdf
```

### ❌ Incorrecto
```
Documento (1).pdf
ficha tecnica.pdf
MANUAL FINAL FINAL (2).PDF
nuevo documento.docx
```

# 🎨 GUÍA DE ILUMINACIÓN Y FONDO

## Para Galería Principal (Fondo Blanco)

### Setup Básico
```
          ☀️ Luz natural (ventana)
           │
           ▼
    ┌──────────────┐
    │   DIFUSOR    │  ← Tela blanca o papel
    └──────────────┘
           │
           ▼
    ┌──────────────┐
    │  PRODUCTO    │
    └──────────────┘
           │
    ───────────────────
    Fondo blanco (cartulina/tela)
```

### Consejos
- ✅ Luz difusa, suave
- ✅ Sin sombras duras
- ✅ Colores fieles al producto real
- ❌ Flash directo
- ❌ Luz amarilla/tungsteno
- ❌ Sombras en una sola dirección

## Para Galería Adicional (Contexto)

Aquí hay más libertad creativa, pero mantén:
- ✅ Producto como protagonista
- ✅ Ambiente ordenado y limpio
- ✅ Buena iluminación (natural preferible)
- ❌ Desorden visual
- ❌ Otros productos que distraigan

# ✅ CHECKLIST FINAL

## Antes de Enviar las Imágenes

### Galería Principal
- [ ] Imágenes en proporción **1:1 (cuadrado)**
- [ ] Resolución mínima **1200 x 1200 px**
- [ ] Fondo **blanco puro**
- [ ] Producto **completo** (no cortado)
- [ ] Producto **centrado** con márgenes
- [ ] Al menos **3 fotos** diferentes ángulos
- [ ] Peso máximo **1MB** por imagen
- [ ] Formato **JPG, PNG o WEBP**

### Galería Adicional
- [ ] Imágenes en proporción **4:3 (horizontal)**
- [ ] Resolución mínima **1200 x 900 px**
- [ ] Imágenes de **contexto/lifestyle** o detalles
- [ ] Peso máximo **500KB** por imagen
- [ ] Formato **JPG, PNG o WEBP**

### Archivos Descargables
- [ ] Formato **PDF** para documentos
- [ ] Nombre descriptivo **sin espacios**
- [ ] Peso máximo **10MB** por archivo
- [ ] Contenido **legible** y profesional

# 🛠️ HERRAMIENTAS ÚTILES

## Para Redimensionar Imágenes
- **Squoosh.app** (gratis, online) - Muy recomendado
- **TinyPNG.com** (gratis, online) - Para comprimir
- **Canva.com** (gratis) - Para recortar a proporciones

## Para Verificar Tamaño
1. Click derecho en la imagen
2. Propiedades (Windows) / Obtener información (Mac)
3. Buscar "Dimensiones"

## Cómo Recortar a 1:1 en tu Celular
1. Abre la foto en la galería
2. Toca "Editar" o ✏️
3. Selecciona "Recortar"
4. Elige proporción "1:1" o "Cuadrado"
5. Ajusta el encuadre (producto centrado)
6. Guarda
